spyice.parameters.algae_constants
===================================

This module defines the **default nutrient and carbon concentrations** for sea ice and underlying water.

Variables
-----------

nutrient_cn_dsi_ice
  - Type: int
  - Default: 15
  - Units: mmol m^-3
  - Description: Initial nutrient concentration in sea ice.

nutrient_cn_dsi_water
  - Type: int
  - Default: 40
  - Units: mmol m^-3
  - Description: Initial nutrient concentration in the underlying water column.

carbon_cc_ice_initial
  - Type: float
  - Default: 0.01
  - Units: mmol m^-3
  - Description: Initial carbon concentration in sea ice.

carbon_cc_water_initial
  - Type: float
  - Default: 0.05
  - Units: mmol m^-3
  - Description: Initial carbon concentration in the underlying water column.
